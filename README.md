# 🦊 치지직 방송 챗봇

**치지직(Chzzk) 플랫폼**을 위한 자동 채팅 및 정보 응답 챗봇입니다.  
Python + Node.js 환경 기반으로, 방송 시작 시 자동 접속하여 채팅 메시지에 반응하며,  
주가/가위바위보/방송 정보 등 다양한 명령어를 제공합니다.

---

## 🛠 기술 스택

- Node.js (buzzk 라이브러리 활용)
- Python (환경 변수 자동 업데이트)
- SQLite (가위바위보 전적 저장)
- cheerio + axios (웹 크롤링)
- bash (자동 실행 스크립트)

---

## ✨ 주요 기능

| 명령어        | 설명                                   |
|---------------|----------------------------------------|
| `!업타임`     | 방송 시작 후 경과 시간 표시            |
| `!팔로우`     | 유저가 방송을 팔로우한 시간 표시       |
| `!주가 <종목명>` | 해당 주식의 현재가/고가/저가 표시     |
| `!가위바위보 <선택>` | 전적을 저장하며 가위바위보 실행 |
| `!홍스피`, `!츄스피` | 현재 시청자 수 표시                 |
| `!방제`       | 현재 방송 제목 출력                    |
| `!명령어`     | 사용 가능한 명령어 목록 출력           |

## 📂 디렉토리 구조

```csharp
dobae/
├── run_buzzk.js # 메인 챗봇 실행 스크립트
├── update_env.py # 환경변수 자동 갱신 (e.g. NID_AUT, NID_SES)
├── run_scripts.sh # 전체 실행 자동화 스크립트
├── package.json # Node.js 의존성 명세
├── requirements.txt # Python 의존성 명세
├── game_results.db # (자동 생성) 가위바위보 전적 저장용 SQLite DB
├── my-custom-buzzk/ # 커스텀 buzzk 모듈 (필요 시)
└── .env # 인증 토큰 보관용 (업로드 금지!)
```

## 🚀 실행 방법

```bash
# 1. 의존성 설치
npm install
pip install -r requirements.txt

# 2. 실행
bash run_scripts.sh
```
